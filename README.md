🔍 Features
🧼 Data Cleaning: Handles nulls, normalizes column names, and converts datetime fields.

🤖 Machine Learning: Trained Random Forest and XGBoost for churn prediction.

💬 NLP: Sentiment analysis and summarization using SpaCy and HuggingFace transformers.

📈 Forecasting: Time series analysis and predictions using Facebook Prophet.

📊 Dashboards: Interactive data visualizations using Apache Superset / React+D3.js.

📝 Reporting: Auto-generated PDF and Excel reports for business stakeholders.
